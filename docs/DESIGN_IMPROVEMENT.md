## 1. 조명 (Lighting)

**임팩트: ★★★★★ | 난이도: 낮음 | 우선순위: 최상**

### 현재 상태

- 글로벌: `ambientLight(0.5)` + `directionalLight(0.4)` 1개
- 방별: 천장 중앙 `pointLight` 1개 (distance=30, decay=2)
- 그림자(Shadow) 미사용

### 문제점

| 항목 | 설명 |
|------|------|
| 작품 조명 부재 | 실제 미술관의 핵심은 작품을 비추는 **스팟라이트**. 현재 천장 중앙 포인트라이트 하나로는 작품이 벽과 같은 밝기로 보여 시선이 집중되지 않음 |
| 그림자 없음 | 프레임이 벽에 그림자를 드리워야 입체감과 깊이가 생기는데, 현재 완전히 평면적 |
| Ambient 비율 과다 | ambient 0.5는 전체 씬을 균일하게 밝혀서 명암 대비가 사라짐. 미술관은 어두운 공간에 작품만 밝은 것이 핵심 |

### 개선안

- [ ] 각 `ArtFrame` 위에 `SpotLight` 추가
  - `angle: 0.4`, `penumbra: 0.5`, `intensity: 1.5`
  - 작품 중심을 target으로 지정
- [ ] `ambientLight` intensity를 `0.15~0.2`로 낮추기
- [ ] `directionalLight`에 `castShadow` 활성화
  - `shadow-mapSize: [1024, 1024]`
- [ ] `ArtFrame` mesh에 `castShadow` / `receiveShadow` 적용
- [ ] 복도/입구는 의도적으로 어둡게 유지 → 방에 들어서면 밝아지는 대비 효과

### 참고 이미지 레퍼런스

```
실제 미술관 조명 패턴:
  [어두운 천장] ──── 트랙 레일 ────
       │              │
    스팟라이트      스팟라이트
       ↓              ↓
    ┌──────┐      ┌──────┐
    │ 작품 │      │ 작품 │
    └──────┘      └──────┘
  ──────── 밝은 벽면 ────────
  ════════ 어두운 바닥 ════════
```

## 2. 재질/텍스처 (Materials)

**임팩트: ★★★★★ | 난이도: 중간 | 우선순위: 최상**

### 현재 상태

- 모든 표면이 `meshStandardMaterial` + 단색(color)만 사용
- 벽: `#f5f0eb`, 바닥: `#2a2a2a` (roughness 0.15, metalness 0.4)
- PBR 텍스처(Normal/Roughness/AO map) 미사용
- Environment Map 미사용

### 문제점

| 항목 | 설명 |
|------|------|
| 바닥이 비현실적 | metalness 0.4로 금속 느낌. 미술관 바닥은 보통 대리석이나 원목인데, 현재 설정은 "젖은 아스팔트"에 가까움 |
| 벽에 질감 없음 | 실제 미술관 벽은 미세한 석고/페인트 텍스처가 있어 빛을 부드럽게 산란시킴 |
| 프레임이 플라스틱 같음 | `#3d2b1f` 단색은 나무 질감이 전혀 느껴지지 않음 |
| 반사/환경맵 없음 | 바닥에 주변 환경이 은은하게 비치면 공간감이 크게 향상됨 |

### 개선안

- [ ] drei의 `<Environment preset="studio" />` 추가 → 모든 재질에 환경 반사 자동 적용
- [ ] **바닥**: 대리석 텍스처 적용
  - diffuse + normal map + roughness map
  - `metalness: 0.1`, `roughness: 0.3`
- [ ] **벽**: 미세한 석고 normal map 적용
  - `roughness: 0.85`
- [ ] **프레임**: 나무결 텍스처 적용
  - diffuse + normal map, `roughness: 0.5`
- [ ] drei의 `<ContactShadows />` 로 바닥 접지감 추가

### 텍스처 소스 추천

| 텍스처 | 무료 소스 |
|--------|-----------|
| 대리석 바닥 | [Poly Haven](https://polyhaven.com/textures) - `white_marble` |
| 석고 벽 | [ambientCG](https://ambientcg.com) - `PaintedPlaster` |
| 나무 프레임 | [ambientCG](https://ambientcg.com) - `WoodFineDark` |

## 3. 포스트프로세싱 (Post-Processing)

**임팩트: ★★★★☆ | 난이도: 중간 | 우선순위: 중간**

### 현재 상태

- 아무런 후처리 효과 없음
- 선형 톤매핑(기본값)

### 문제점

- 미술관의 고급스러운 분위기를 만드는 데 후처리가 필수
- 작품 조명의 은은한 glow 효과가 없으면 조명이 날카롭게 보임

### 개선안

`@react-three/postprocessing` 패키지 도입:

- [ ] **SSAO** (Screen Space Ambient Occlusion)
  - 모서리/구석에 자연스러운 그림자 → 공간 깊이감
  - `radius: 0.1`, `intensity: 1.5`
- [ ] **Bloom**
  - 밝은 조명 주변 은은한 광채
  - `threshold: 0.9`, `intensity: 0.3`, `luminanceSmoothing: 0.9`
- [ ] **Vignette**
  - 화면 가장자리 어둡게 → 시네마틱 느낌
  - `darkness: 0.4`, `offset: 0.3`
- [ ] **ToneMapping**
  - ACESFilmic → 사실적인 HDR 톤 매핑

### 성능 고려

```typescript
// 모바일에서는 후처리 비활성화 또는 경량화
const isMobile = useIsMobile();

<EffectComposer enabled={!isMobile}>
  <SSAO />
  <Bloom />
  <Vignette />
</EffectComposer>
```

## 4. 건축 디테일 (Architectural Details)

**임팩트: ★★★★☆ | 난이도: 중간 | 우선순위: 중간**

### 현재 상태

- 모든 방이 6면 BoxGeometry (바닥/천장/벽 4면)
- 문은 벽에 구멍만 뚫린 형태
- 장식적 요소 전무

### 문제점

| 항목 | 설명 |
|------|------|
| "마인크래프트 방" 느낌 | 직사각형 박스만으로는 건축적 매력이 없음 |
| 문틀 없음 | 문 개구부에 프레임이 없어 벽이 잘린 것처럼 보임 |
| 몰딩/베이스보드 없음 | 벽과 바닥/천장이 만나는 지점이 날카롭게 끊김 |
| 천장 디테일 없음 | 실제 미술관은 코니스, 조명 레일 등이 있음 |

### 개선안

- [ ] **문틀 (Door Frame)**
  - 문 개구부 주변에 얇은 BoxGeometry 프레임
  - 두께 `0.08m`, 깊이 `0.15m`
- [ ] **걸레받이 (Baseboard)**
  - 벽 하단에 높이 `0.1m` 돌출 띠
  - 벽보다 약간 어두운 색상
- [ ] **크라운 몰딩 (Crown Molding)**
  - 천장-벽 접합부에 장식 라인
- [ ] **천장 조명 레일**
  - 스팟라이트가 달린 트랙 조명 시각적 표현
  - 얇은 BoxGeometry 레일 + 원통형 조명 기구
- [ ] **입구 아치**
  - Entrance Hall 입구에 아치형 개구부

### 구조 다이어그램

```
크라운 몰딩 ─────────────────────────
│                                    │
│   ┌─────────────────────────────┐  │
│   │         천장 조명 레일        │  │
│   │    ◉         ◉         ◉   │  │
│   └─────────────────────────────┘  │
│                                    │
│   ┌────┐                ┌────┐    │
│   │    │                │    │    │  벽면
│   │작품│                │작품│    │
│   │    │                │    │    │
│   └────┘                └────┘    │
│                                    │
걸레받이 ─────────────────────────────
═══════════ 바닥 (대리석) ═══════════
```

## 5. 작품 프레임 (ArtFrame)

**임팩트: ★★★★☆ | 난이도: 중간 | 우선순위: 높음**

### 현재 상태

- 단순 BoxGeometry 프레임 (두께 0.03m)
- 캔버스는 평면 PlaneGeometry
- InfoPlate: 작은 평면 + 텍스트

### 문제점

| 항목 | 설명 |
|------|------|
| 프레임에 깊이감 없음 | 실제 액자는 안쪽으로 단이 있는(stepped) 프로파일 |
| 매트(여백) 없음 | 작품과 프레임 사이의 흰색 여백(mat)이 없어 싸구려 보임 |
| 유리 반사 없음 | 실제 액자의 유리 느낌이 없음 |
| 빈 작품 placeholder 없음 | 이미지가 없을 때 빈 베이지색 평면만 보임 |

### 개선안

- [ ] **2단 프로파일 프레임**
  - 외곽 프레임 + 내곽 프레임 단차
  - 외곽: `0.06m` 폭, 내곽: `0.03m` 폭, 단차: `0.01m`
- [ ] **매트 (Mat) 레이어**
  - 작품과 프레임 사이 흰색 여백
  - `#fafafa`, 폭 `0.04m`
- [ ] **유리 효과 레이어**
  - `meshPhysicalMaterial`
  - `transmission: 0.95`, `roughness: 0.05`, `ior: 1.5`
- [ ] **빈 작품 placeholder**
  - 우아한 gradient 또는 미니멀한 아이콘 패턴
- [ ] **InfoPlate 개선**
  - 은은한 backlight 효과 (emissive)
  - 살짝 올라온 3D 플레이트 (두께 `0.005m`)

### 프레임 단면도

```
현재:
  ┌────────────────────┐
  │ ┌────────────────┐ │
  │ │                │ │  ← 단일 프레임 + 캔버스
  │ │     작품       │ │
  │ │                │ │
  │ └────────────────┘ │
  └────────────────────┘

개선 후:
  ┌──────────────────────┐  ← 외곽 프레임 (나무 텍스처)
  │ ┌──────────────────┐ │
  │ │ ┌──────────────┐ │ │  ← 내곽 프레임 (단차)
  │ │ │ ┌──────────┐ │ │ │
  │ │ │ │          │ │ │ │  ← 매트 (흰색 여백)
  │ │ │ │   작품   │ │ │ │
  │ │ │ │          │ │ │ │  ← 유리 레이어 (반투명)
  │ │ │ └──────────┘ │ │ │
  │ │ └──────────────┘ │ │
  │ └──────────────────┘ │
  └──────────────────────┘
```

## 6. 안개/분위기 (Atmosphere)

**임팩트: ★★★☆☆ | 난이도: 낮음 | 우선순위: 낮음**

### 현재 상태

- 선형 Fog: start 30m, end 80m, color `#1a1a1a`
- Sparkles: 입구에만 40개 파티클

### 문제점

| 항목 | 설명 |
|------|------|
| 안개 시작이 너무 멂 | 30m 시작이면 대부분의 실내 공간에서 안개가 보이지 않음 |
| 분위기 파티클이 입구에만 제한 | 나머지 방은 시각적 효과가 없음 |

### 개선안

- [ ] 안개 범위 조정: `start: 15m`, `end: 50m`
  - 먼 방이 자연스럽게 사라지는 효과
- [ ] 각 방에 미세한 먼지 파티클 추가
  - drei `<Sparkles count={15} opacity={0.15} />`
- [ ] Wing 테마별 파티클 색상 차별화
  - Grand: 금빛, Warm: 오렌지, Cool: 하늘색, Dark: 보라, Cozy: 크림

## 7. 카메라 & 시야 (Camera & FOV)

**임팩트: ★★☆☆☆ | 난이도: 낮음 | 우선순위: 낮음**

### 현재 상태

- FOV: 70도
- 작품 포커스 시 눈높이 2.55m에서 관람

### 문제점

| 항목 | 설명 |
|------|------|
| FOV 70은 약간 좁음 | 실내 공간에서 75~80도가 더 자연스러움 |
| 포커스 높이 고정 | 작품 중심 높이에 따라 카메라 높이가 달라져야 함 |

### 개선안

- [ ] FOV `75`로 조정 (실내 건축 시각화 표준)
- [ ] 포커스 카메라 높이를 작품 중심 높이에 맞추기 (고정값 → 동적 계산)
- [ ] 포커스 전환 시 약간의 DOF (Depth of Field) 블러 효과
  - `@react-three/postprocessing`의 `<DepthOfField>`
  - `focusDistance: 0.02`, `focalLength: 0.05`, `bokehScale: 3`

## 8. 바닥 가이드 (Floor Guide)

**임팩트: ★★★☆☆ | 난이도: 중간 | 우선순위: 낮음**

### 현재 상태

- 대시 라인 + 콘 화살표
- emissive 색상으로 발광 효과

### 문제점

- 게임 UI 같은 발광 화살표는 미술관 분위기와 이질적
- 몰입감을 깨뜨림

### 개선안

- [ ] **매립형 LED 스트립 스타일**로 변경
  - 얇은 발광 라인 (width: `0.01m`)
  - 낮은 intensity (emissive: `0.3`)
  - 바닥 표면에 매립된 느낌 (Y: `0.001`)
- [ ] 또는 **바닥 타일 패턴 변화**로 자연스럽게 동선 유도
- [ ] 화살표 대신 바닥에 각인된 방 이름/아이콘

## 핵심 요약

> **Phase 1만 적용해도 (1~2시간 작업) "실제 미술관을 걷는 느낌"으로 한 단계 도약할 수 있습니다.**
>
> 현재 프로젝트는 기능적 완성도가 높지만, 시각적으로는 모든 표면이 단색 + 균일한 조명으로
> 인해 "3D 프로토타입" 느낌입니다. 조명의 명암 대비와 환경 반사만 추가해도
> 공간의 깊이감과 고급스러움이 극적으로 달라집니다.
